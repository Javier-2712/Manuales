Taller2.2 Intro_Matrices

Resumen de las actividades

Objetivo de la actividad

El presente taller tiene como objetivo, realizar un entrenamiento b√°sico sobre algebra lineal aplicada a datos multivariados. Iniciar√©mos con operaciones b√°sicas (taller 2.2) y posteriormente se realizar√°n algunas aplicaciones estad√≠sticas como el an√°lisis de componentes principales (pca - taller 2.2a) y el an√°lisis discriminante (lda - taller 2.2b).

Talleres a realizar en clase

Son tres los talleres a realizar:

  1. Taller 2.2 Operaciones matriciales b√°sicas. El objetivo de esta actividad es adquirir destrezas en operaciones b√°sicas con matrices que sirven de soporte para diferentes dise√±os multivariados.

  2. Taller 2.2a. Operaciones matriciales para an√°lisis de componentes principales - pca. Se aplicar√°n operaciones matriciales, mediante un procedimiento paso a paso, para la realizaci√≥n de esta t√©cnica de ordenaci√≥n multivariada - pca.

  3. Taller 2.2b. Operaciones matriciales para el an√°lisis discriminante lineal - lda. Se aplicar√°n operaciones matriciales, mediante un procedimiento paso a paso, para la realizaci√≥n de esta t√©cnica de clasificaci√≥n multivariada - lda.

7. Funci√≥n para automatizar el lda

La siguiente funci√≥n, permite ahorrar varios pasos, porque discrimina a todos los estudiantes (observaciones o filas de la base de datos), haya los vectores y la funci√≥n discriminante de cada estudiante y realiza las tabla que compara a la clasificaci√≥n inicial por sexos vs. la clasificaci√≥n del modelo del discriminante lineal - lda. Esta funci√≥n resume a todo el paso 7 del ejercicio anterior.

# Funci√≥n discriminante para todos los estudiantes
lda <- datos2[,2:5] %>%
  as.matrix() %>%
  apply(1, function(x) {
    x <- as.matrix(x)  # Convertir x a un vector columna (4x1)
    W <- (t(dif) %*% cov.g.i %*% x) - (1/2 * t(dif) %*% cov.g.i %*% sum)
    return(W)
  }) %>%
   tibble(Alumnos = rownames(datos2), resultados_W = .) %>%        # Redondear a 2 decimales
  mutate(Sexo_discriminado = ifelse(resultados_W < 0, "F", "M"),  # Discriminante
         Sexo_original = datos2$Sexo)  # A√±adir columna con el sexo original

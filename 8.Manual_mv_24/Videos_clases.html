Pautas para estudios de caso

Los siguientes videos muestran el procedimiento para la organización de una base de datos de molúscos terrestres. Enfatizando en tres partes (1) la forma como se pueden resumir taxones para que no saturen las figuras exploratorias, (2) la implementación de técnicas multivariadas que incorporan a todos los taxones y su relación con variables ambientales y (3) Generar variables agrupadoras (o factores), para poder visualizar gradientes espaciales o temporales. Esto último, para los casos en que no contemos con esas variables. Finalmente se orienta cómo generar el reporte en html con el paquete quarto.

  • 🎥 Pautas para el procesamiento de estudio de caso - muluscos
  • 🎥 Pautas para el procesamiento de estudio de caso - mariposas

El siguiente video muestra cómo utilizar el repositorio de GitHub (enlace) para generar reportes HTML y conectarlos a la versión en línea de R, llamada posit cloud (enlace). Este ejercicio se realizará con el ejemplo del CLA de bahías (2024-I)

  • 🎥Pautas para construir un HTML en GitHub y vincularlo en posit-cloud (RStudio en línea).

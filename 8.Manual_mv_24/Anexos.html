Bases de datos requeridas en algunos talleres

Nota: Para descargar estos archivos hay que hacer clic derecho sobre este y Guardar enlace como

Taller 2 Introducción al RStudio y a Matrices en RStudio

  1. (datos1.csv) Base de datos con información de 4 variables morfométricas de estudiantes de Unimagdalena.

Taller 3. Figuras Exploratorias

  1. (Insectos.xlsx) Base de datos con con información de dos variables ambientales y con órdenes de insectos en diferentes quebradas y en dos cuencas.

  2. (Insectos.csv) Base de datos similar a la anterior pero en formato csv.

  3. (Insectos2.csv) Base de datos con información de Abundancia (Ab) y de Biomasa (Biom) de invertebrados, por sitios,por muestreos, Por grupos funcionales (GF) y por periodos climáticos.

Taller 4. Análisis de Componentes Principales - PCA

  1. (FQmarino.csv) Base de datos con información de 7 variables ambientales y 7 bahías de Santa Marta.

  2. (Tayrona.csv) Base de datos con información de diferentes taxones de invertebrados y de variables ambientales.

Taller 5. Análisis de Escalamiento Multidimensional - NMDS

  1. (malezas.csv) Base de datos con información de taxones de malezas en diferentes lugares del Magdalena.

Taller 6. Análisis de Clúster - CLA

  1. (FQmarino1.csv) Base de datos con información de 7 variables ambientales y 7 bahías de Santa Marta.

  2. (datos.csv) Base de datos con información morfométrica de individuos pertenecientes a 5 especies de peces de la India.

Tallerers 7 al 9. Análisis Discrimianante -LDA, Manovas y Permanovas

  1. (peces.csv) Base de datos identica a la anterios del CLA, con información morfométrica de individuos pertenecientes a 5 especies de peces de la India.
